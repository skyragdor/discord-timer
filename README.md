# Discord Timer

Маленька утиліта для Windows що відстежує твою активність у Discord і показує:
- ⏱ таймер з часу останнього відправленого повідомлення
- 🕐 поточний час
- 💬 скільки повідомлень ти надіслав за сесію
- 📋 лог останніх 5 повідомлень з паузами між ними

Вікно завжди поверх інших програм і не заважає роботі.

## Вимоги

- Windows 10 / 11
- [AutoHotkey v2](https://www.autohotkey.com/) — безкоштовно, легко встановити

---

## Встановлення

1. Встанови **AutoHotkey v2** з [autohotkey.com](https://www.autohotkey.com)
2. Скачай `DiscordTimer.ahk` з цього репозиторію
3. Двічі клікни по файлу — програма запуститься

---

## Як користуватись

- Запусти програму — вікно з'явиться у правому верхньому куті екрану
- Відкрий Discord (додаток або браузер) і напиши повідомлення
- Після першого **Enter** таймер автоматично стартує
- Кожне нове повідомлення скидає таймер на нуль

### Кнопки

| Кнопка | Дія |
|---|---|
| Пауза / Продовжити | Зупиняє або відновлює таймер |
| Скинути | Скидає таймер на `00:00:00` |
| Очистити лог | Скидає все — таймер, лічильник, лог |

### Кольори таймера

| Колір | Значення |
|---|---|
| 🟢 Зелений | менше 2 хвилин |
| 🟡 Жовтий | від 2 до 5 хвилин |
| 🔴 Червоний | більше 5 хвилин |

## Як додати до автозапуску

Щоб програма стартувала разом з Windows:

1. Натисни `Win + R`, введи `shell:startup`
2. Скопіюй `DiscordTimer.ahk` у відкриту папку
